# V6 → V6.1 单项选择流程更新

## 修正内容

首页先让用户选择一个项目：

- 摩托车选择推荐
- 头盔
- 手套
- 骑行服/护具
- 骑行靴
- 三箱
- 射灯
- 蓝牙耳机
- 防盗/盗抢保障

用户选择哪一项，就只完成哪一项。不会自动要求把全部项目做一遍。

## 覆盖更新

把更新包内文件按原目录覆盖V6项目，然后执行：

```bash
npm run check
npm run build
npm run dev
```

## 验收

1. 首页应显示9个项目入口。
2. 点“摩托车选择推荐”进入12轮选车。
3. 点“手套”先显示提醒图，再直接进入手套测试。
4. 手套结果页点击“完成，返回项目选择”，不得自动进入头盔或其他项目。
5. 其他装备类目同理。
6. 按U进入装备单项选择页。
