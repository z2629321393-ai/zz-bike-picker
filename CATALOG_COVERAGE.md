因此，1,482 不能直接叫作“1,482 款摩托车”。网站展示的是“可审计的公开两轮索引快照”，不是“全国全部可买可上牌摩托车”。车型类别、上市、停售、库存和地方登记政策都要继续核验，任何永久性的“全”都不可验证。

当前 23 条都是车型族初核，不是 23 个已核准具体版本。逐项证据和名称拆分见 [`NEW_LISTING_AUDIT_2026-07-17.md`](NEW_LISTING_AUDIT_2026-07-17.md) 与 [`UPCOMING_UNQUOTED_AUDIT_2026-07-17.md`](UPCOMING_UNQUOTED_AUDIT_2026-07-17.md)：当前全部保持 `recommendable: false`。第二批还确认 AE4i、Fz5 110 属于电动自行车，不能混进摩托车推荐数量。

## 两个状态必须分开

1. **道路准入状态**：具体版本是否能同时闭环中国大陆工信部产品/进口记录、有效 CCC 与一致性依据。
2. **市场供应状态**：核验日是否仍有品牌中国官方销售、订购/询价或正规授权库存渠道。

工信部公告中有记录不等于目前仍在销售；第三方有指导价也不等于经销商有货，更不等于任一城市都能登记。

## 正式车型建议状态

道路准入：

```text
verified_domestic_road
verified_official_import
approval_pending
non_road
no_cn_road_evidence
document_abnormal
```

市场供应：

```text
official_on_sale
official_inquiry
preorder
stock_only
discontinued_used_only
no_official_channel_found
availability_unknown
```

正常新车推荐必须同时满足：具体版本、中国大陆工信部记录、有效 CCC、官方在售/可询价、证据未过期、参数无冲突、人工审核通过。停售、未引进、即将上市、纯赛道/非道路、手续异常及过期待复核记录不得进入默认推荐。

“水车”通常描述某一辆实车的来源或手续，而不是车型名称。同一车型可能同时存在正规官方进口车和非正规渠道车辆，必须核对 VIN、发票、合格证/进口凭证、CCC/一致性证书等单车材料；泡水事故车则属于二手车车况问题。

## 来源优先级

1. [工信部道路机动车辆生产企业及产品信息查询系统](https://service.miit-eidc.org.cn/miitxxgk/gonggao_xxgk/index_ggcp.html)；[工信部官方说明](https://www.miit.gov.cn/jgsj/zbys/gzdt/art/2019/art_0dad606713eb4430b110161999462773.html)。
2. [国家认监委摩托车 CCC 规则](https://www.cnca.gov.cn/zwxx/gg/2021/art/2022/art_faf1952d6fb542ceb8798f612c34dace.html)与[证书查询入口](https://cx.cnca.cn/CertECloud/index/index/page)。
3. [《机动车登记规定》](https://www.moj.gov.cn/pub/sfbgw/flfggz/flfggzbmgz/202305/t20230509_478410.html)，用于核对登记材料与不予登记情形。
4. 品牌中国官网、官方商城、官方订购/询价和授权经销网络，用于确认当前市场供应。
5. 摩托范等第三方公开目录只负责发现候选、别名和历史参考价，不能单独批准进入推荐。

## 接近“全量”的维护办法

- 每次工信部新公告后做增量比对。
- 每周比对品牌中国官网、价格、下订入口和车型上下架。
- 市场供应证据建议 30–90 天后自动标为待复核。
- 爬虫只创建 `pending` 候选，禁止自动进入正式推荐库。
- 每条记录对应“品牌 + 车系 + 具体配置 + 年款/批次 + 公告型号”，不同排量与配置不能再合并共用参数。
- 地方禁限摩、牌照额度和迁入政策另行维护；最终以当地车管所与交管部门当日答复为准。

只有完成所有已发现品牌与具体版本的逐款差集核对后，页面才可以写“截至某日，本次盘点范围内已核验”，仍不能写永久性的“国内所有车型”。
